# School images — upload guide

All website photos for static pages live under `public/school-images/`.

**URL path:** `/school-images/...`  
Example: file `public/school-images/student-life/annual-sports/photo-01.jpg` → `/school-images/student-life/annual-sports/photo-01.jpg`

## Folder layout

| Folder | Use for |
|--------|---------|
| `site-wide/hero-sliders/` | Homepage hero carousel slides |
| `site-wide/marquee/` | Homepage “Discover Playpen” marquee cards |
| `site-wide/page-heroes/` | Shared banner images for section pages |
| `about/` | About pages (campus, administration, alumni, career, etc.) |
| `academics/` | Academic section pages and school levels |
| `admissions/` | Admissions pages |
| `student-life/` | Sports, science fair, cultural programme, ECA, services, etc. |
| `gallery/` | Photo gallery (see below) |
| `_incoming-unsorted/` | Temporary drop zone — sort into the right folder later |

## Gallery uploads

Gallery images are grouped by category. **Create a new subfolder for each event:**

```
gallery/
  sports/annual-sports-2026/
  celebrations/graduation-2025/
  arts/cultural-programme-2025/
  custom-events/your-new-event-2027/   ← any new event
```

**Naming tips**

- Use lowercase and hyphens: `annual-sports-2026`, `science-fair-2025`
- Include the year when it is an annual event
- Put all photos for one event in one folder
- Use simple file names: `01-opening.jpg`, `02-race.jpg`

## File tips

- Prefer `.jpg` or `.webp` for photos; `.png` for graphics with transparency
- Keep originals; compress before upload if files are very large (aim under ~500 KB per web photo when possible)
- Do not use spaces in folder or file names

## Adding a new section later

1. Create a new folder under the matching section (e.g. `student-life/new-event-name/`)
2. Upload images there
3. Tell the web team the folder path so the page can use them
